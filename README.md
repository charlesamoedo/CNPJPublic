# 🚀 CNPJ Public BR

Aplicação web leve para **consulta rápida de CNPJ** utilizando dados públicos.

> 🔎 Ideal para profissionais das áreas fiscal, contábil, comercial e integrações que precisam validar informações de empresas de forma ágil.

---

## 🧠 Problema que resolve

No dia a dia, consultar dados de empresas no Brasil pode ser:

- Lento
- Dependente de sistemas pesados
- Pouco prático para consultas rápidas

Este projeto foi criado com um objetivo simples:

> ⚡ **Fornecer uma forma rápida, direta e leve de consultar dados de CNPJ**

---

## 🌐 Demonstração

Acesse a aplicação online:

👉 https://cnpj-public-br.vercel.app/

---

## ⚙️ Funcionalidades

- 🔍 Consulta de CNPJ em tempo real
- 📄 Exibição de dados públicos da empresa
- 💻 Interface simples e objetiva
- ⚡ Aplicação leve e rápida
- 📦 Possibilidade de instalação (PWA)

---

## 🏗️ Arquitetura

Estrutura atual do projeto:

```
Frontend (Web App)
  ├── Interface de consulta
  ├── Consumo de API pública
  └── Renderização dos dados
```

### Tecnologias utilizadas

- JavaScript / TypeScript
- Framework frontend (ex: React)
- Vercel (deploy)
- API pública de dados de CNPJ

---

## 🔌 Fonte dos dados

Os dados são obtidos a partir de **APIs públicas de consulta de CNPJ**.

> ℹ️ Este projeto não armazena dados — apenas consome e exibe informações públicas.

---

## ▶️ Como rodar o projeto localmente

```bash
# Clone o repositório
git clone https://github.com/charlesamoedo/CNPJPublic.git

# Acesse a pasta
cd CNPJPublic

# Instale as dependências
npm install

# Execute o projeto
npm run dev
```
